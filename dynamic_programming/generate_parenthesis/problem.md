# Generate Parentheses

Given `n` pairs of parentheses, 
write a function to generate all combinations of well-formed parentheses.

### Sample Input #1
```
n = 3
```
### Sample Output #1
```
((())),  (()()), (())(), ()(()), ()()()
```

### Sample Input #2
```
n = 1
```
### Sample Output #2
```
()
```
